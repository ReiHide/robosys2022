# robosys2022
ロボットシステム学の練習リポジトリ

## sumコマンド
* 標準入力から読み込んだ数字の和、平方の和、平方根の和、自然対数の和を計算し、出力するコマンド


## インストール方法
~~~
git clone https://github.com/ReiHide/robosys2022.git
~~~

## 実行例
* 入力例

~~~
　seq 2 4 | ./sum
~~~
* 出力例

~~~
　9.0　29.0　5.146264369941973　3.1780538303479453
~~~
## テスト環境
* Ubuntu20.04

## 必要ソフトウェア
* Python
  * テスト済み: Python 3.7～3.10
